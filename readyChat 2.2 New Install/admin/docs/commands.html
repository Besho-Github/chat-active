    • 1) Chat Commands

    readyChat's commands allow moderators and administrators to control the chat room easily and remove trouble makers quickly.

    /kick username

    The kick command allows you to disconnect a user from the chat room. They'll be able to log straight back in again, but as a first warning this can prove useful. To kick a user, the correct syntax is "/kick username". The user will be disconnected instantly and shown a kicked message.
    /ban username
    The ban command is similar to the kick command. The difference is the user will not be able to log in after becoming a banned user. To ban a user, the correct syntax is "/ban username". The user will be disconnected instantly and shown a banned message.
    /warn username message
    Warning a user will present them with a message on screen. This is a great way to grab someone's attention when they don't seem to be reading your messages. To warn a user, the correct syntax is "/warn username message".
    /prune
    The prune command will clear the messages from the room you're currently in. To clear messages from all rooms, the Global Maintenance tool would be more efficient. the correct syntax is "/prune".
