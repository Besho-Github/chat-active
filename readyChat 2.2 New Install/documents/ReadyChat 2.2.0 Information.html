    Thank you for purchasing readyChat. Version 2.2.0 includes several new features (primarily tool additions) and several fixes for the readyChat software.

    2) Compatibility

    ReadyChat 2.2.0 is compatible with 2.2.0 and includes an upgrade tool which will create new database entries for you. Please follow the instructions in the "update" folder.

    A list of changed files can be found in the update notes which you should review before uploading the files.
